Deployment Guide

Add this tag just before </body> in your HTML. Replace the highlighted values with your credentials.

<!-- AI Cyber SOC — Real-Time Threat Monitor -->
<script
  src="https://your-soc.onrender.com/static/agent.js"
  data-site="YOUR_SITE_ID"
  data-key="YOUR_API_KEY"
  defer
></script>

Add to functions.php in your active theme, or use a plugin like "Insert Headers and Footers".

// In functions.php
function soc_monitor_script() {
  echo '<script src="https://your-soc.onrender.com/static/agent.js"
    data-site="YOUR_SITE_ID"
    data-key="YOUR_API_KEY"
    defer></script>';
}
add_action('wp_footer', 'soc_monitor_script');

Add to your _app.js or layout.js (App Router).

/* In _app.js or layout.js */
import Script from 'next/script'

export default function Layout({ children }) {
  return (
    <html>
      <body>
        {children}
        <Script
          src="https://your-soc.onrender.com/static/agent.js"
          data-site="YOUR_SITE_ID"
          data-key="YOUR_API_KEY"
          strategy="afterInteractive"
        />
      </body>
    </html>
  )
}

Add before the closing </body> tag in your footer template.

<?php /* In footer.php or your template */ ?>
<script
  src="<?= htmlspecialchars('https://your-soc.onrender.com/static/agent.js') ?>"
  data-site="YOUR_SITE_ID"
  data-key="YOUR_API_KEY"
  defer
></script>

Add to your base template base.html before the closing </body>.

{# In your Django/Jinja2 base template #}
<script
  src="https://your-soc.onrender.com/static/agent.js"
  data-site="YOUR_SITE_ID"
  data-key="YOUR_API_KEY"
  defer
></script>

Go to the SOC Dashboard → Site Monitor tab → click "+ Add New Site". Enter your website's name and URL. A unique Site ID and API Key will be generated.
2
Copy Your Credentials
From the site card above, copy your Site ID and API Key. These are unique to your website and ensure your data is completely isolated from other organisations.
3
Embed the Agent Script
Select your platform above and paste the code snippet into your website's HTML. The agent is < 5 KB, loads asynchronously, and will never slow down your site.
4
Verify Deployment
Click "Send Test Event" below. Within 5 seconds, a test event should appear in your Site Monitor tab on the dashboard. You're live! 🎉
